# Chess

Desktop chess project for playing against an AI opponent.

Current repository contents:
- Python chess engine with legal move generation, notation, tests, and learning-memory support
- In-progress desktop/web front-end files and launch scaffolding
- Persistent game record folder for saved matches

Core files:
- `chess_engine.py`
- `chess_popup.py`
- `chess_popup_server.py`
- `test_chess_engine.py`

Run tests:

```powershell
python -m unittest -v test_chess_engine.py
```
